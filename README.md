# 🍔 MERN Food Delivery App

A full-stack food ordering and delivery web application built using the MERN stack.  
It allows users to browse food items, add to cart, place orders, and manage authentication.

---

---

## 🧠 Tech Stack

**Frontend:**
- React.js (Vite)
- CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Local / Atlas)

**Other Tools:**
- Mongoose
- JWT Authentication
- CORS
- Dotenv

---

## ✨ Features

- 👤 User Signup & Login (JWT Authentication)
- 🍕 Browse Food Items
- 🛒 Add to Cart / Remove Items
- 📦 Place Orders
- 🧾 Order Management
- 🔐 Secure API routes
- ⚡ Fast Vite frontend
